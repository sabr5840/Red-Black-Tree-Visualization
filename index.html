The 5 Basic Rules for red-black trees

  • Root Color:The root of the tree must always be black.
  • No Two Red Notes Together: A red node must not have a red parent.
  • Black Height: Each path from a node to its null points (leaves) must have the same number of black nodes. Red notes are not counted in this height.
  • Null Points:All null points (leaves) are black.
  • New Nodes Are Red: When you insert a new node, it is colored red by default.

Left and Right Rotations

  • Left Rotation:
    • Purpose: To balance the tree by moving one node down to the left and another node up.
    • Action:If a node has a right child, this right child becomes the new root, and the original node becomes the left child of the new root.
  • Right Rotation:
    • Purpose:To balance the tree by moving one node down to the right and another node up.
    • Action: If a node has a left child, this left child becomes the new root, and the original node becomes the right child of the new root.
